## 2. Glossário de Propriedades Principais
* **color:** [Explique que serve para mudar a cor da fonte/texto]
* **background-color:** [Explique que serve para mudar a cor de fundo do elemento]
* **margin:** [Explique que é o espaço do lado de FORA do elemento, empurrando os outros para longe]
* **padding:** [Explique que é o espaço do lado de DENTRO do elemento, entre a borda e o conteúdo]
* **display: flex:** [Explique que serve para organizar os elementos na tela de forma flexível, lado a lado ou em coluna]

## 3. Como as "Classes" ajudam
[Explique que as classes (usadas com um ponto no CSS, ex: .minha-classe) servem para dar um "apelido" para um elemento específico no HTML. Assim, você consegue estilizar apenas aquele elemento sem alterar os outros iguais a ele no site.]
